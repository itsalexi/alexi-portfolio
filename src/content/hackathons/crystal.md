## What it is

Crystal is an ongoing StartupQC project about shared team context.

The idea is simple: teams make decisions, write things down, change plans, and move fast. A lot of useful context gets scattered along the way. Crystal explores what it would feel like if that context stayed easier to find and understand.

## Why we’re building it

Small teams already carry a lot in their heads. As more people join, the same questions come back, old decisions become harder to trace, and new members need a faster way to catch up.

Crystal is our attempt at making that shared memory feel more usable.

## Where it is now

Crystal is currently being built for **StartupQC Squad 3**, where we recently got in as a **finalist**.

The project is still ongoing, so this page is a snapshot rather than a finished write-up.
