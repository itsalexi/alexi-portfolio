Delivered an introductory workshop on Git and GitHub for incoming CS students, covering fundamental concepts of version control and collaborative software development.

## Workshop Content

- Git basics: commits, branches, and merges
- GitHub workflow and pull requests
- Collaborative development best practices
- Resolving merge conflicts

## Impact

Students completed hands-on exercises including creating their first repository, making commits, and submitting pull requests. This workshop helped establish a strong foundation for their future development projects.
